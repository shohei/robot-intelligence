＜C言語で記述されたプログラムのコンパイルについて＞

　掲載しておりますプログラムは，Visual C++.netにて動作確認を行っております．

　Visual C++.net を使用してコンパイルする場合には，プロジェクトのテンプレートとして「Win32コンソールプロジェクト」をご使用ください．
　また，必要に応じて，各ソースコードの先頭に「#include "stdafx.h"」を追加してください．


　他のコンパイラを使用される場合にも，コンソールアプリケーションがコンパイルできるよう必要な設定を行ってください．


＜正誤情報＞
１．ソースコード５・４
	先頭に以下の二行を追加してください．
	#include <stdio.h>
	#include <time.h>

２．ソースコード５・６
	1行目の#include"stdafx.h"は，Visual C++.netを使用してコンパイルするときのみ必要です．他のコンパイラを使用される場合には削除してください．
